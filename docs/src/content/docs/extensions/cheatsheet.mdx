Press `ctrl+?` or run any of the command aliases to print the full
list of keyboard shortcuts inline. The output uses a responsive
multi-column layout that adapts to terminal width.

## Commands

| Command | Description |
|---------|-------------|
| `/cheatsheet` | Show keyboard shortcuts |
| `/keys` | Alias for `/cheatsheet` |
| `/keymap` | Alias for `/cheatsheet` |
| `/keybindings` | Alias for `/cheatsheet` |
